# Validation & KPI (Phase 2)

## 📋 Checklist de Validation du Noyau
- [ ] Ouverture de l'application après suppression du cache.
- [ ] Premier lancement sans données (comportement par défaut).
- [ ] Export JSON des données.
- [ ] Suppression manuelle des données.
- [ ] Import JSON.
- [ ] Vérification que toutes les données reviennent.
- [ ] Navigation sans erreur sur tous les onglets.
- [ ] Utilisation continue pendant plusieurs heures (Focus Mode).
- [ ] Recharge (F5) de la page.
- [ ] Fermeture complète puis réouverture du navigateur.

## 📊 KPI du Produit (À mesurer en Phase 2)
1. **Temps d'ouverture de l'application** : ms
2. **Temps de génération du planning** : ms
3. **Temps passé réellement en Focus** : minutes/jour
4. **Taux de tâches terminées** : %
5. **Nombre de jours de streak** : jours
6. **Temps perdu (prévu vs réel)** : delta en minutes
